# Physical Count Module

> Complete inventory verification through systematic counting of all items in selected locations.

## Overview

The Physical Count module enables comprehensive inventory verification by counting all items across selected locations. It provides a structured 4-step wizard for count setup, real-time counting interface, and variance analysis.

## Key Features

- **4-Step Creation Wizard**: Setup → Location Selection → Item Review → Final Review
- **Dashboard**: KPIs, activity charts, and counts management
- **Active Counting**: Real-time item counting with status tracking
- **Variance Analysis**: Automatic variance detection and categorization
