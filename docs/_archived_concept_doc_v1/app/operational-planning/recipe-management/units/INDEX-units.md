## Overview

The **Recipe Units** module manages measurement units used in recipes, supporting unit conversions and ensuring consistent ingredient quantities across all recipe operations.

**Key Features**:
- Standard unit definitions (kg, g, L, ml, ea, etc.)
- Unit type classification (weight, volume, count)
- Conversion factor management
- Base unit relationships
- Active/inactive status
- Unit abbreviations and display names
