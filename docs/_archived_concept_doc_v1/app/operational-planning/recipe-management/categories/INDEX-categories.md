## Overview

The **Recipe Categories** module enables organization of recipes into logical groups for easier navigation, menu planning, and reporting. Categories support hierarchical structures allowing parent-child relationships for detailed classification.

**Key Features**:
- Hierarchical category structure (parent/child)
- Active/inactive status management
- Sort order customization
- Category-based reporting
- Integration with menu engineering
