## Overview

The **Cuisine Types** module enables classification of recipes by their culinary origin or style. This supports menu diversity analysis, guest preference tracking, and cuisine-based reporting.

**Key Features**:
- Cuisine origin classification (Italian, Asian, Mediterranean, etc.)
- Active/inactive status management
- Description and characteristics capture
- Recipe count tracking per cuisine
- Cuisine-based filtering and reporting
