# Use Cases: Notification Preferences

## Module Information
- **Module**: System Administration
- **Sub-Module**: Notification Preferences
- **Route**: `/system-administration/settings/notifications`
- **Version**: 1.0.0
- **Last Updated**: 2026-01-17

---

## UC-NP-001: Configure Global Defaults

**Actor**: IT Administrator

**Main Flow**:
1. Navigate to Notification Preferences
2. Select Defaults tab
3. Expand event category
4. Toggle event enabled/disabled
5. Select notification channels
6. Set frequency option
7. Click Save Changes

**Alternative Flows**:
- Bulk enable: Toggle "Enable All" for category
- Reset: Click Reset to Default

---

## UC-NP-002: Edit Email Template

**Actor**: IT Administrator

**Main Flow**:
1. Select Templates tab
2. Click template from list
3. View template details
4. Click Edit
5. Modify subject/body
6. Save changes

---

## UC-NP-003: Send Test Email

**Actor**: IT Administrator

**Main Flow**:
1. Select template
2. Click Send Test
3. Confirm recipient
4. System sends test email
5. Toast confirms sent

---

## UC-NP-004: View Notification History

**Actor**: IT Administrator

**Main Flow**:
1. Select History tab
2. View notification list
3. Apply filters (date, type, status)
4. Click entry for details

---

## UC-NP-005: Create Routing Rule

**Actor**: IT Administrator

**Main Flow**:
1. Select Routing tab
2. Click Add Rule
3. Select event type
4. Define conditions
5. Select recipients
6. Save rule

---

## UC-NP-006: Test Notification

**Actor**: IT Administrator

**Main Flow**:
1. Select Testing tab
2. Choose notification type
3. Select channel
4. Enter test recipient
5. Click Send Test
6. Verify delivery
